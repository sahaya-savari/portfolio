## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/sahaya-savari/portfolio.git
cd portfolio
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Production Version

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

---

## 📦 Deployment

Deploy to Firebase Hosting:

```bash
npm run build
npx firebase deploy
```
