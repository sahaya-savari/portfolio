# Sahaya Savari F — Portfolio

A modern, glassmorphic, hardware-accelerated personal portfolio website built to showcase my projects, skills, and background in Artificial Intelligence.

## Overview
I am an AI Developer currently pursuing my M.Sc in Artificial Intelligence. This portfolio highlights my practical AI applications, backend systems, and data-driven projects in a highly interactive, premium UI format.

## Tech Stack
- **Framework**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, GSAP
- **PDF Rendering**: react-pdf
- **Media Delivery**: hls.js (HTTP Live Streaming)

## Features
- 🚀 **Hardware-Accelerated Animations**: Liquid-smooth scrolling, entering, and transitioning powered by Framer Motion.
- 🎨 **Custom Glassmorphism Design System**: Clean, futuristic, and premium aesthetic.
- 📄 **In-Page Premium Resume Viewer**: Custom lazy-loaded PDF viewer with zoom and pagination controls.
- 🔍 **Deep Dive Project Modals**: Interactive modals detailing technical metrics, focus areas, and direct links.
- 📱 **Fully Responsive**: Flawless experience across desktop, tablet, and mobile.
- ♿ **Accessible**: Semantic structure with universal `ESC` key bindings, focus trapping, and ARIA labels.

## Installation

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/sahaya-savari/portfolio.git
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Build Instructions

To create a production-ready build:

```bash
npm run build
```

This will generate a highly optimized, code-split bundle (or a single-file build depending on Vite configuration) inside the `dist/` directory.
